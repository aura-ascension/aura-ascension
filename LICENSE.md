## 🔧 Common Issues and Fixes

**❌ GUI doesn't appear after execution**
→ Press **Right Shift** to manually toggle the GUI. If it still doesn't show, clear your executor's cache, restart Roblox, and re-execute the script from scratch.

**❌ Auto train not switching zones**
→ The smart zone detection requires your character to be in a recognized training area on startup. Walk into any training zone manually, then toggle **Smart Auto Train** off and back on to re-initialize.

**❌ Auto rebirth not triggering**
→ Check **Settings > Auto Rebirth > Threshold** and confirm it's set correctly for your current power tier. After major game updates, thresholds may need manual recalibration.

**❌ Aura sniper not finding rare auras**
→ Increase the scan radius in **Settings > Aura Sniper** and confirm the **Rare Only** filter is disabled if you want all aura tiers detected. Rare aura spawn rates in the game itself are low — this is expected behavior.

**❌ Script flagged by antivirus**
→ This is a standard false positive with Lua injector executors. Add your executor's folder to your antivirus exclusion list. The script is open-source and contains no malicious code.

**❌ Mobile version crashing on injection**
→ Make sure you're using the **Android variant** of the **aura ascension script roblox** paste, clearly labeled in the raw file header. PC and mobile builds use different GUI rendering systems.

**❌ Anti-AFK not preventing kick**
→ Enable anti-AFK *immediately* after script execution, before going idle. The module must intercept the idle counter before it starts — enabling it mid-idle session won't reset the timer.
